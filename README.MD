## 技术选型
    ● 核心框架：Spring Boot 1.5.1.RELEASE
    ● 安全框架：Spring Security
    ● 任务调度：Spring + Quartz

    ● 日志管理：logback
    ● 前端框架：Metronic + Bootstrap + Jquery
